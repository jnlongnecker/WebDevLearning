This example uses several callback functions together in order to function:

  • One callback function starts the timer after a delay for the animation.
  • One callback function stops the timer after the time specified.
  • One callback function decrements the timer.
  • One callback function ends the animation after the time specified.
  • One callback function enables the button after the animation completes.

Additionally, there is exception handling for invalid inputs. Try a bad number and see what happens.
