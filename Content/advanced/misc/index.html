Weird Truths about JavaScript

Primitives vs Non-Primitives

The following are Primitive Data Types

  • Boolean
  • Null
  • Undefined
  • Number
  • BigInt
  • String
  • Symbol

The following are Non-Primitive Data Types

  • Object
  • Function (inherits from Object)
  • Array (inherits from Object)
  • Map
  • Set
  • JSON
  • More! (Anything Custom)

In JavaScript, primitives are passed by value while non-primitives are passed by reference. This demo will add 5 to each number after being passed to another function that displays after a delay. The only difference is the right number is wrapped in an Object, so it gets passed by reference.
